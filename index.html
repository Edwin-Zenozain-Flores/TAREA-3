Investigación y aplicación de metodologías BEM y camelCase

Esta página web fue diseñada como un proyecto informativo para explicar y comparar dos metodologías de nombrado muy utilizadas en desarrollo web: BEM y camelCase. A través de secciones claras, listas y ejemplos prácticos en HTML, CSS y JavaScript

BEM

¿Qué es BEM?

BEM (Bloque, Elemento, Modificador) es una metodología para nombrar clases en CSS. Su objetivo es organizar el código de forma más clara, ordenada y fácil de mantener. Gracias a su estructura, es especialmente útil en proyectos grandes o en trabajo en equipo.

Ventajas de usar BEM

  1. Hace que el código HTML y CSS sea más fácil de leer y entender.
  2. Reduce conflictos entre estilos al mantener cada clase en su propio contexto.
  3. Fomenta la creación de componentes reutilizables y modulares.
  4. Facilita el mantenimiento y crecimiento del proyecto con el tiempo.
  5. Ayuda a que todo el equipo use la misma convención de nombres.

Ejemplos de código usando BEM

  • tarjeta: Bloque principal para una tarjeta de contenido.
  • tarjeta__imagen: Elemento que representa la imagen dentro de la tarjeta.
  • tarjeta__imagen--grande: Modificador que indica una versión más grande de la imagen.
  • formulario__campo: Elemento que representa un campo dentro del formulario.
  • formulario__campo--invalido: Modificador que marca el campo como con error.

camelCase

Explicación de qué es camelCase

camelCase es una forma de escribir palabras juntas sin espacios, donde la primera palabra empieza en minúscula y cada palabra siguiente comienza con mayúscula, como las jorobas de un camello.

Por ejemplo

nombreUsuario

calcularEdad

listaDeProductos

¿Para que se usa?

Se usa principalmente en programación para nombrar variables, funciones, objetos y métodos, porque hace que los nombres sean fáciles de leer y entender.

¿Por qué es util?

Evita espacios o caracteres especiales en los nombres.

Hace que los nombres largos sean más claros y legibles.

Es una convención muy común y ayuda a que el código sea consistente.

Lista ordenada de situaciones donde se usa camelCase

  1. let nombreUsuario = "Edwin"; — Variable
  2. function calcularTotal() { } — Función
  3. persona.calcularPrecio(); — Método de objeto
  4. producto.precioTotal — Propiedad de objeto
  5. const listaDeCompras = []; — Variable (estructura de datos)

Lista desordenada de ejemplos de código usando camelCase

  • let nombreCompleto = "Edwin Zenozain"; — Variable
  • const productosEnCarrito = 2; — Variable
  • function mostrarMensajePersonalizado() { } — Función
  • const productosEnCarrito = ["producto1", "producto2"]; — Variable (array)
  • usuario.obtenerDatos(); — Método de objeto

¿En qué casos conviene usar BEM o camelCase?

El enfoque BEM es ideal para el desarrollo de interfaces web en HTML y CSS, sobre todo en situaciones como:

  • Proyectos donde es importante mantener una estructura visual clara y modular.
  • Cuando se busca evitar conflictos entre clases CSS similares o duplicadas.
  • Entornos colaborativos donde varios desarrolladores trabajan sobre los mismos archivos.
  • Casos donde se necesita mantener un código escalable, reutilizable y fácil de depurar.

Por otro lado, el estilo camelCase es más común en JavaScript y otros lenguajes de programación, y suele aplicarse principalmente en:

  1. Nombres de variables y constantes (por ejemplo: totalPrecio).
  2. Funciones o métodos que representan acciones (por ejemplo: calcularTotal()).
  3. Propiedades de objetos y nombres de eventos personalizados.
  4. Proyectos donde se combina JavaScript con frameworks como React o Vue.
<div class="perfil perfil--activo">
 <h2 class="perfil__nombre">Edwin Zenozain</h2>
 <p class="perfil__biografia">Estudiante apasionado por el desarrollo web.</p>
</div>

Utiliza camelCase cuando trabajes con JavaScript, especialmente en:

  1. Nombres de variables que almacenan datos.
  2. Funciones que realizan acciones o cálculos.
  3. Propiedades y métodos dentro de objetos.
1-variable que almacena un dato
    let usuarioActivo = "Lucía";
  function obtenerEdadActual(anioNacimiento) {
    return 2025 - anioNacimiento;
  }

  2-funcion que realiza un calculo y devuelve un resultado
  const productosEnCarrito = ["laptop", "mouse", "teclado"];

  3-objeto con propiedades y metodos
  const perfilUsuario = {
    nombreCompleto: "Lucía Ramírez",
    correoElectronico: "lucia@mail.com",
    iniciarSesion: function() {
      console.log("Sesión iniciada");
    }
  };
